Toejam & Earl Seeded World Generator

What is this?

This is the Toejam & Earl Seeded World Generator! It takes the original Toejam & Earl game for the Sega Genesis, and applies new seeds. This allows different players to race each other using consistent seeds.

This is a web-based patching system for Sega Genesis/Mega Drive ROMs; the patching is done entirely in your web browser using the power of JavaScript. (No ROMs are uploaded or downloaded in this process.) In order to use it, you will need a ROM for Toejam & Earl. If you do not have such a ROM, and you are on Windows, you can purchase it from Steam as one of the SEGA Mega Drive and Genesis Classics. Once installed, the file you need is at <Steam Directory>\SteamApps\common\Sega Classics\uncompressed ROMs\ToeJamEarl.SGD.

The ROM data will be stored in your web browser's offline storage for future use.

How does it work?

The base game includes both a "Random World" game and a "Fixed World" game. The "Fixed World" uses predefined values for all the level seeds and ship piece locations. We merely patch in our own values, and then rewrite the menu so that "Fixed World" becomes "Seeded World", so that the maps and ship piece locations will be consistent between runs. Note that, just like in a Fixed World game, the present appearances and earthling locations are still randomized.

But then can't someone cheat by forcing a good seed?

The main menu is modified to include both the seed identifier and the version of the generator. Most places where someone would care about "cheating"-- that is, races and the speedrun.com leaderboards-- require streaming or video evidence, including the menu at game start.

Seeded World runs should not be considered as valid for submission for either the Fixed World or Random World leaderboards.

What version of the game do I need?

For this system, either REV 00 or REV 02 is acceptable. (The version from Steam is REV 02.)

There are no known gameplay differences between the two versions.

What do the options do?

Level Types

Original
The original mix of level types. Level 4 is a big lake, Level 22 is a big desert, etc.
Shuffled
Level types are shuffled around.
Randomized
Any level may be one of eight different level types.

Earthling Locations

Original
Vanilla Earthling placement.
Shuffled
Earthling groups are shuffled between levels. You might get Level 22's ice cream trucks on Level 2. Surprise!
Randomized
The Earthlings from level 2 through 25 are redistributed (mostly) randomly. Only at most one good earthling of each type per level.

Starting Inventory

Original
Vanilla starting inventory (Bonus Hitops)
Mystery
Start with four mystery boxes.
Random
Start with four random unidentified presents.
Random Bonus
Replace the "Bonus Hitops" with a new bonus present.
